* ## Data Understanding
Tahap ini berfokus pada eksplorasi dan pemahaman karakteristik data.

### Tujuan:
* Menilai kualitas data
* Mengidentifikasi pola awal
* Mendeteksi anomali

### Langkah Detail:
#### 1. Pengumpulan Data
* Sumber data bisa berupa:
* Database internal
* API
* File CSV/Excel
* Data publik

#### 2. Deskripsi Struktur Data
* Jumlah baris dan kolom
* Tipe data (numerik, kategorikal, datetime)
* Distribusi nilai

#### 3. Exploratory Data Analysis (EDA)
Dilakukan dengan:
* Statistik deskriptif (mean, median, std)
* Visualisasi (histogram, boxplot, heatmap)
* Korelasi antar variabel
#### 4. Identifikasi Masalah Data
* Missing values
* Outliers
* Duplikasi
* Inkonsistensi format
Output tahap ini:
* Insight awal
* Laporan kualitas data
* Hipotesis awal
