Proyek Missing Values Inputation + Materi Normalisasi data#

Catatan ini membahas dua topik utama dalam pengolahan data, yaitu penanganan missing values dan normalisasi data. Missing values adalah nilai yang hilang dalam dataset sehingga perlu dilakukan imputasi untuk melengkapi data. Selain itu, dibahas juga normalisasi data, yaitu proses menyamakan skala nilai antar atribut agar data lebih mudah dan akurat untuk dianalisis.

Missing values#

Pengertian Missing Values#

Missing values adalah kondisi ketika sebagian data dalam dataset tidak memiliki nilai atau kosong. Biasanya ditandai dengan:

  • kosong (blank)

  • NA

  • NULL

  • NaN

Missing values sering terjadi dalam proses pengumpulan data, misalnya:

  • responden tidak mengisi suatu pertanyaan

  • kesalahan sistem saat menyimpan data

  • data tidak tersedia

Jika missing values tidak ditangani, maka dapat menyebabkan:

  • analisis data menjadi tidak akurat

  • algoritma machine learning tidak dapat berjalan

  • hasil penelitian menjadi bias

Karena itu diperlukan proses imputasi data.

Imputasi Missing Values#

Imputasi adalah proses mengisi nilai yang hilang dengan nilai perkiraan berdasarkan informasi dari data lain dalam dataset. Beberapa metode imputasi yang sering digunakan:

  1. Mean (rata-rata)

  2. Median

  3. Modus

  4. KNN Imputation

  5. WKNN (Weighted K-Nearest Neighbor)

Pada tugas ini digunakan metode WKNN.

Metode Mengatasi Missing Values: WKNN#

Weighted K-Nearest Neighbor (WKNN) adalah metode yang digunakan untuk mengisi missing value dengan memanfaatkan data yang paling mirip atau paling dekat dengan data yang memiliki nilai kosong. Metode ini bekerja dengan cara mencari beberapa tetangga terdekat (K-Nearest Neighbor) berdasarkan jarak antar data, kemudian nilai yang hilang diperkirakan menggunakan rata-rata berbobot dari tetangga tersebut.

Dalam metode WKNN, setiap tetangga diberikan bobot berdasarkan jaraknya, di mana data yang jaraknya lebih dekat akan memiliki bobot lebih besar dibandingkan data yang lebih jauh. Bobot biasanya dihitung menggunakan rumus kebalikan jarak (1/d). Dengan demikian, nilai yang lebih dekat akan memberikan pengaruh lebih besar dalam proses estimasi missing value.

Metode WKNN sering digunakan karena mampu menghasilkan estimasi yang lebih akurat dibandingkan metode sederhana seperti mengganti nilai dengan rata-rata, karena WKNN mempertimbangkan kemiripan antar data dalam proses pengisian nilai yang hilang.

Menghitung Jarak (Euclidean Distance)#

Rumus Jarak: $\(d(i,8) = \sqrt{(HR_i - HR_8)^2 + (Sleep_i - Sleep_8)^2 + (Cal_i - Cal_8)^2}\)$

Contoh perhitungan data S1: $\(\begin{aligned} d &= \sqrt{(0.314 - 0.445)^2 + (0.510 - 0.478)^2 + (0.362 - 0.494)^2} \\ d &= \sqrt{0.01716 + 0.00102 + 0.01742} \\ d &= 0.189 \end{aligned}\)$

Hasil ini sesuai dengan nilai jarak S1 pada tabel yaitu 0.189.

Lakukan Proses perhitungan seterusnya sampai S15, kecuali S8 karena tidak diukur jaraknya terhadap dirinya sendiri.

Menghitung Bobot#

Setelah jarak diperoleh, bobot dihitung dengan rumus: $\(w_{i} = \frac{1}{d_{i}}\)$

Contoh S1: $\(\begin{aligned} w &= \frac{1}{0.189} \\ w &= 5.299 \end{aligned}\)$

Semakin kecil jarak, maka bobot semakin besar.

Menghitung Nilai Imputasi#

Rumus: $\(w_{i} \times x_{i}\)$

Contoh S1: $\(5.299 \times 0.424 = 2.247\)$

Hasil tersebut sesuai dengan nilai pada tabel.

Melakukan Perangkingan Jarak#

Setelah nilai jarak dan bobot diperoleh, langkah berikutnya adalah melakukan perangkingan berdasarkan nilai jarak terkecil. Semakin kecil jarak suatu data terhadap data target (data ke-8), maka data tersebut dianggap semakin mirip.

Menghitung Jarak Menggunakan Euclidean Distance#

Kode ini mendefinisikan sebuah fungsi untuk menghitung jarak Euclidean antara dua vektor numerik. Fungsi ini akan digunakan untuk mengukur kemiripan antara data target dan setiap data lain dalam proses WKNN.

def euclidean(a, b):
    return np.sqrt(np.sum((a - b) ** 2))

Penjelasan Kode:

  • def euclidean(a, b): : Mendefinisikan fungsi bernama euclidean yang menerima dua input, yaitu vektor a dan b.

  • return np.sqrt(np.sum((a - b) ** 2)) : Menghitung jarak Euclidean:

    • (a - b) : selisih tiap elemen vektor a dan b.

    • ** 2 : mengkuadratkan setiap selisih.

    • np.sum(…) : menjumlahkan seluruh kuadrat selisih.

    • np.sqrt(…) : mengambil akar kuadrat dari jumlah tersebut untuk mendapatkan jarak Euclidean.

Kesimpulan Missing Values dengan Metode WKNN#

Missing value merupakan kondisi ketika terdapat data yang hilang atau tidak terisi pada suatu atribut dalam dataset. Salah satu metode yang dapat digunakan untuk mengatasi masalah ini adalah Weighted K-Nearest Neighbor (WKNN). Metode WKNN bekerja dengan mencari beberapa data yang paling mirip atau paling dekat dengan data yang memiliki nilai hilang, kemudian nilai tersebut diperkirakan menggunakan rata-rata berbobot dari tetangga terdekat.

Dalam WKNN, data yang memiliki jarak lebih dekat akan diberikan bobot yang lebih besar, sehingga memiliki pengaruh lebih besar dalam menentukan nilai pengganti. Metode ini dapat digunakan pada berbagai tipe data dan mampu memberikan hasil yang lebih akurat karena mempertimbangkan kemiripan antar data, bukan hanya menggunakan rata-rata sederhana. Dengan demikian, WKNN menjadi salah satu metode yang efektif untuk mengatasi missing value dalam proses pengolahan data.

Normalisasi data#

Pengertian Normalisasi#

Normalisasi data adalah proses mengubah skala nilai data menjadi rentang tertentu tanpa mengubah informasi dasar dari data tersebut. Normalisasi biasanya digunakan dalam pengolahan data dan machine learning agar setiap atribut memiliki pengaruh yang seimbang dalam proses analisis.

Dalam sebuah dataset sering terdapat atribut yang memiliki skala nilai berbeda. Misalnya suatu dataset memiliki atribut jarak tempuh kendaraan, konsumsi bahan bakar, dan kapasitas mesin. Nilai dari atribut tersebut bisa memiliki rentang yang sangat berbeda. Jika data tersebut digunakan langsung dalam algoritma seperti KNN atau clustering, maka atribut dengan nilai lebih besar akan memiliki pengaruh lebih besar dalam perhitungan jarak.

Oleh karena itu, normalisasi dilakukan agar semua atribut berada dalam skala yang sebanding sehingga proses analisis menjadi lebih adil dan akurat.

Macam-Macam Normalisasi#

A. Min-Max Normalization#

Min-Max Normalization adalah metode normalisasi yang mengubah nilai data ke dalam rentang tertentu, biasanya antara 0 sampai 1. Metode ini bekerja dengan cara mengurangi nilai data dengan nilai minimum kemudian membaginya dengan selisih antara nilai maksimum dan minimum.

Tujuan dari metode ini adalah agar seluruh nilai data berada pada skala yang sama tanpa mengubah hubungan antar data.

Rumus Min-Max Normalization:

\[x' = \frac{x - min(x)}{max(x) - min(x)}\]

Keterangan:

  • \(x\) = nilai data asli

  • \(min(x)\) = nilai minimum dalam dataset

  • \(max(x)\) = nilai maksimum dalam dataset

  • \(x'\) = nilai hasil normalisasi

Contoh Data

10

Nilai minimum = 2

Nilai maksimum = 10

Perhitungan

  • Mahasiswa A: \((2 − 2) / (10 − 2) = 0\)

  • Mahasiswa B: \((5 − 2) / (10 − 2) = 0.37\)

  • Mahasiswa C: \((7 − 2) / (10 − 2) = 0.62\)

  • Mahasiswa D: \((10 − 2) / (10 − 2) = 1\)

Metode ini sangat sering digunakan dalam algoritma berbasis jarak seperti KNN dan K-Means karena membuat skala data menjadi seragam.

B. Z-Score Normalization (Standardization)#

Z-score normalization atau standardization adalah metode normalisasi yang mengubah data berdasarkan rata-rata (mean) dan standar deviasi dari dataset.

Metode ini menghasilkan data dengan:

  • rata-rata = 0

  • standar deviasi = 1

Rumus Z-score:

\[z = \frac{x - \mu}{\sigma}\]

Keterangan:

  • \(x\) = nilai data

  • \(\mu\) = rata-rata data

  • \(\sigma\) = standar deviasi

Contoh Data

Dataset waktu belajar mahasiswa per hari (jam):

Mahasiswa

Jam Belajar

A

2

B

4

C

6

D

8

Rata-rata : \(5\)

Standar deviasi

  • Cari Rata-rata (\(\bar{x}\))

\[\bar{x} = \frac{2 + 4 + 6 + 8}{4} = \frac{20}{4} = 5\]

  • Hitung Selisih Kuadrat dari Rata-rata kurangi setiap angka dengan rata-rata, lalu kuadratkan:

    • \((2 - 5)^2 = (-3)^2 = 9\)

    • \((4 - 5)^2 = (-1)^2 = 1\)

    • \((6 - 5)^2 = (1)^2 = 1\)

    • \((8 - 5)^2 = (3)^2 = 9\)

  • Total Jumlah Kuadrat (\(\sum\)):

\[9 + 1 + 1 + 9 = 20\]

  • Hitung Varian \(s^2\) Karena ini dianggap sebagai satu kelompok data (populasi kecil), kita bagi totalnya dengan jumlah data (\(n = 4\)):

\[s^2 = \frac{20}{4} \approx 5\]

  • Hitung Standar Deviasi (\(s\)) Akar kuadrat dari varian: $\(s = \sqrt{5} \approx 2,236\)$

Perhitungan

  • Mahasiswa A: \((2 - 5) / 2,236 = -1,342\)

  • Mahasiswa B: \((4 - 5) / 2,236 = -0,447\)

  • Mahasiswa C: \((6 - 5) / 2,236 = 0,447\)

  • Mahasiswa D: \((8 - 5) / 2,236 = 1,342\)

C. Decimal Scaling Normalization#

Decimal Scaling adalah metode normalisasi yang dilakukan dengan cara membagi nilai data dengan pangkat 10 tertentu sehingga nilai data menjadi lebih kecil.

Rumus:

Keterangan:

  • \(x\) = nilai data asli

  • \(j\) = jumlah digit dari nilai terbesar dalam dataset

Implementasi Normalisasi Menggunakan (Library Sklearn) Python#

A. Min-Max Normalization menggunakan library sklearn#

import numpy as np
from sklearn.preprocessing import MinMaxScaler
import pandas as pd

data = {
    'Mahasiswa': ['A', 'B', 'C', 'D'],
    'Buku Dibaca': [2, 5, 7, 10]
}
df = pd.DataFrame(data)

X = df[['Buku Dibaca']].values
scaler = MinMaxScaler()
df['Normalisasi'] = scaler.fit_transform(X)

print("Hasil Min-Max Normalization:")
print(df)

C. Decimal Scaling Normalization menggunakan Function#

import numpy as np
import pandas as pd
from sklearn.preprocessing import FunctionTransformer

data = {
    'Minggu': [1, 2, 3, 4],
    'Pengunjung': [250, 300, 450, 500]
}
df = pd.DataFrame(data)

max_val = df['Pengunjung'].abs().max()
k = len(str(int(max_val)))

def decimal_scaler(X):
    return X / (10**k)

scaler = FunctionTransformer(decimal_scaler)

df['Hasil Normalisasi'] = scaler.transform(df[['Pengunjung']])

print(f"Nilai k yang digunakan: {k} (Pembagi: {10**k})")
print("\nTable Hasil Decimal Scaling:")
print(df)

  • import numpy as np : digunakan untuk memanggil library NumPy untuk operasi numerik.

  • import pandas as pd : digunakan untuk memanggil library Pandas untuk mengolah data dalam bentuk tabel (DataFrame).

  • from sklearn.preprocessing import FunctionTransformer : digunakan untuk mengimpor FunctionTransformer yang memungkinkan kita membuat fungsi normalisasi sendiri.

  • data berisi dataset yang terdiri dari minggu dan jumlah pengunjung.

  • pd.DataFrame(data) : mengubah data menjadi DataFrame agar mudah diproses.

  • decimal_scaler(X) : adalah fungsi yang membagi setiap nilai data dengan 10^k agar nilainya berada pada skala kecil.

  • FunctionTransformer(decimal_scaler) : membuat objek scaler berdasarkan fungsi yang telah dibuat.

  • scaler.transform(df[[‘Pengunjung’]]) : menerapkan fungsi Decimal Scaling pada data pengunjung.

  • df[‘Hasil Normalisasi’] : menyimpan hasil normalisasi ke kolom baru.

  • print() : digunakan untuk menampilkan nilai k yang digunakan serta tabel hasil normalisasi data.

Kesimpulan Normalisasi data#

Normalisasi data adalah proses mengubah nilai data ke dalam skala tertentu agar perbedaan antar nilai tidak terlalu besar dan lebih mudah dianalisis. Proses ini sering digunakan dalam pengolahan data dan machine learning agar setiap atribut memiliki skala yang sebanding. Beberapa metode normalisasi yang umum digunakan adalah Min-Max Normalization, Z-Score Normalization, dan Decimal Scaling. Dengan melakukan normalisasi, data menjadi lebih konsisten, perhitungan menjadi lebih stabil, dan hasil analisis atau model yang dihasilkan menjadi lebih akurat.
